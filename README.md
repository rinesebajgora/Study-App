## Description
Ky është një aplikacion për studim me AI (StudyAI) që ndihmon përdoruesit të bëjnë pyetje dhe të marrin përgjigje inteligjente për tema të ndryshme. Përdoruesit mund të:

- regjistrohen dhe login
- bëjnë pyetje tek AI
- ruajnë përgjigjet
- përditësojnë dhe fshijnë pyetjet
- organizojnë pyetjet sipas subject-it
- përdorin dark mode

---

## Technologies
- Next.js
- React
- Supabase (Auth + Database)
- Tailwind CSS
- Vercel (Deployment)

---

## Features
- Authentication (Login/Signup)
- AI Q&A system
- CRUD për pyetje
- Organizim sipas subject-it
- Dark Mode (ruhet në localStorage)
- UI responsive dhe moderne
